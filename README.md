# Job Application Form using Streamlit

A simple and professional job application form built using **Python** and **Streamlit** with **email automation using SMTP**.

## Features
- User-friendly web form
- Name, email, password input
- Gender selection
- Position dropdown
- Skills multiselect
- Terms and conditions checkbox
- Automatic confirmation email
- Form validation

## Technologies Used
- Python
- Streamlit
- SMTP
- Gmail App Password
